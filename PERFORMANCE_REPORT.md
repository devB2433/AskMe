# AskMe系统性能优化报告

- **CPU**: 16核心

## 性能测试结果

### 1. 查询处理性能
- **平均处理时间**: 2.56 毫秒
- **性能评级**: 优秀 ✅

- **模型**: sentence-transformers/all-MiniLM-L6-v2
- **向量维度**: 384

- **加载时间**: ~0.5 秒

- **Milvus连接**: 成功
